Game Logic

Here is a list of the logic flow

  1. Player Selects a Valid Piece (coded already)
  2. Player Selects a new location for said piece (partially coded)
  3. Move is validated based on team/piece (not coded yet)
  4. Perform a capture if necessary (nest in the move validator function)
  5. Move the actual piece (coded already)
  6. Determine weather a check/checkmate is reached
